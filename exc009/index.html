Marcação

Podemos criar um texto marcado utlizando a tag mark:

<mark>texto<mark>

E para alterar a cor do mark é só utilizar a tag abaixo:

<mark style="background-color:"cor";>texto<mark>

Texto excluído e incluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado através dessa tag:

<del>texto<del>

Como esse exemplo aqui.


E também podemos marca um texto como incluído através dessa tag:

<ins>texto<ins>

Como esse exemplo aqui.




Texto sobrescrito e subscrito

Para usar texto sobrescrito (2 10) é necessário utilizar essa tag:

<sup>texto<sup>


E para usar um texto subscrito (H 2O) a tag é a:

<sub>texto<sub>
