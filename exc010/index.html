Outras formatações

Code

a tag abaixo é utilizada para inserir fonte de textos mono espaçados (usa para dar sentido de código):

<code>texto<code>

Exemplo: Usada para fazer o texto ficar assim.


Pré

Usada para deixar o texto 100% como está escrito no html

<pre>texto<pre>

Exemplo:

<style>
        mark {
            background-color: chartreuse; 
        }
</style>


Citações

Para fazer citações basta utilizar a tag quotes como está abaixo:

<q>texto<q>

ou

<blockquote>texto<blockquote>

Exemplo 1: Como diria o pai de um amigo: o computador é um burro muito rápido.

Exemplo 1:

Como diria Albert Einstein:

Os computadores são incrivelmente rápidos, precisos e burros; os homens são incrivelmente lentos, imprecisos e brilhantes; juntos, seus poderes ultrapassam os limites da imaginação.


Abreviações

Para usar abreviatura utilize a tag abaixo:

<abbr>texto<abbr>

Exemplo: Estou amando HTML e CSS.


Texto invertido

Para inserir texto invertido, utilizar a tag abaixo:

<bdo dir="rtl">texto</bdo>

Exemplo:Como diria o pai de um amigo: o computador é um burro muito rápido
