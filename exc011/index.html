Listas

Listas Ordenadas

para você fazer uma lista ordenada basta utilizr a tag abaixo:

<ol>
<li>texto</li>
<ol>

Exemplo:

  1. Acordar
  2. tomar café
  3. escovar os dentes

Listas não Ordenadas

Para fazer uma lista não ordenada pasta utilizar essa tag abaixou:

<ul>
<li>texto</li>
<ul>
  • leite
  • manteiga
  • pão

OBS: Com <ul type="estilo" você pode selecionar o tipo da lista que você quer
