Como criar parágrafos e linhas:

Para criar um parágrafo basta só adicionar a tag <p> (paragraph):

<p>texto</p>

E para adicionar uma quebra de linha pasta utilizar no final do texto a tag <b> (break):

<p>texto<br>

OBS: A tag break não tem chave de fechamento como a paragraph (</p>).


Como usar simbolos especiais e emojis:

Para criar caracteres especiais como ®, ©, ™, €, $, £, ¢, Δ, Ω, ↑ basta utilizar essa tag &+codigo; :

&+reg; para "®"

&+copy; para "©"

&+trade; para "™"

E para usar emoji você vai utilizar a tag &#x+código; :

&#x+1F596; para "🖖"

&#x+1F60F; para "😏"

&#x+1F44A; para "👊"


Site para obter código de caracteres:
www.w3schools.com/charsets/ref_utf_symbols.asp

Site para obter código de emoji:
www.emojipedia.org

OBS: Não usar o "+" na hora de inserir a tag.

Para adicionar o ícone de favorito basta só inserir a tag abaixo em cima do "<title>...</title>":
