Como criar parágrafos e linhas:

Para criar um parágrafo basta só adicionar a tag <p> (paragraph):

<p>texto</p>

E para adicionar uma quebra de linha pasta utilizar no final do texto a tag <b> (break):

<p>texto<br>

OBS: A tag break não tem chave de fechamento como a paragraph (</p>).


Como usar simbolos especiais e emojis:

Para criar caracteres especiais como ®, ©, ™, €, $, £, ¢, Δ, Ω, ↑ basta utilizar essa tag &+codigo; :

&+reg; para "®"

&+copy; para "©"

&+trade; para "™"

E para usar emoji você vai utilizar a tag &#x+código; :

&#x+1F596; para "🖖"

&#x+1F60F; para "😏"

&#x+1F44A; para "👊"


Site para obter código de caracteres:
www.w3schools.com/charsets/ref_utf_symbols.asp

Site para obter código de emoji:
www.emojipedia.org

OBS: Não usar o "+" na hora de inserir a tag.

O que é semântica?

A semântica é a área da linguística que estuda o significado e a sua relação com o significante. O significado está associado ao sentido e, portanto, ao conteúdo e ao contexto; o significante está associado à forma.

O HMTL5 é focado em semântica (significado) diferente do CSS que é focado em forma e estilo.

Algumas tags com o tempo se tornam obsoletas (como exemplo a tag <body bgcolor="">), por isso sempre é bom você ficar ligados nas atualizaçõs que a W3C volta e meia faz. Segue a baixo alguns exemplos de tags obsoletas

OBS: caso a tag fique vermelha no Visual Studio siginifica que a tag está obsoleta, ou seja não se usa mais essa tag.


Como usar as tags para Negrito e Itálico?

Negrito/Destaque:

A tag correta para usar o negrito é a <strong>texto<strong>

OBS: É recomendado não utilizar a utilizar a tag bold (<b>texto<b>), pois ela não tem semântica.


Itálico/Ênfase:

A tag correta para usar o itálico é a <em>texto<em>

OBS: É recomendado não utilizar a tag italic (<i>texto<i>), pois ela também não tem semântica.


Marcação

E para alterar a cor do mark é só utilizar a tag abaixo:

Para usar texto sobrescrito (2 10) é necessário utilizar essa tag:

Listas

Listas Ordenadas

para você fazer uma lista ordenada basta utilizr a tag abaixo:

    <ol>
    <li>texto</li>
    <ol>

Exemplo:

  1. Acordar
  2. tomar café
  3. escovar os dentes

Listas não Ordenadas

Para fazer uma lista não ordenada pasta utilizar essa tag abaixou:

    <ul>
    <li>texto</li>
    <ul>
  • leite
  • manteiga
  • pão

OBS: Com <ul type="estilo" você pode selecionar o tipo da lista que você quer.


Usando links Externos

Para usar link externo utilize a tag Anchor:

<a href="link do site">descrição</a>


Para abrir o link em outra aba basta só adicionar isso em sua tag:

<a href="link do site" target = "_blank" rel = "external">descrição</a>
