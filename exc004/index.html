Ícone da página (favicon)

Como fazer o comando?

Para adicionar o ícone de favorito basta só inserir essa tag no head:

<link rel ="endereço do ícone" href="html.ico" type="image/x-icon">

EX:

Sites para obter icones:

Para buscar ícone: iconarchive

Para criar ícone: favicon.io

Para desenhar ícone: favicon.cc
