Ícone da página (favicon)

Como fazer usar a tag?

Para adicionar o ícone de favorito basta só inserir essa tag no head acima do "<title>...</title>":

<link rel ="shortcut icon" href="endereço do ícone" type="image/x-icon">

EX:

Sites para obter icones:

Para buscar ícone: iconarchive

Para criar ícone: favicon.io

Para desenhar ícone: favicon.cc
