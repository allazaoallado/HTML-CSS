Semântica HTML5

O que é semântica?

A semântica é a área da linguística que estuda o significado e a sua relação com o significante. O significado está associado ao sentido e, portanto, ao conteúdo e ao contexto; o significante está associado à forma.

O HMTL5 é focado em semântica (significado) diferente do CSS que é focado em forma e estilo.

Algumas tags com o tempo se tornam obsoletas (como exemplo a tag <body bgcolor="">), por isso sempre é bom você ficar ligados nas atualizaçõs que a W3C volta e meia faz. Segue a baixo alguns exemplos de tags obsoletas

OBS: caso a tag fique vermelha no Visual Studio siginifica que a tag está obsoleta, ou seja não se usa mais essa tag.


Como usar as tags para Negrito e Itálico?

Negrito/Destaque:

A tag correta para usar o negrito é a <strong>texto<strong>

OBS: É recomendado não utilizar a utilizar a tag bold (<b>texto<b>), pois ela não tem semântica.


Itálico/Ênfase:

A tag correta para usar o itálico é a <em>texto<em>

OBS: É recomendado não utilizar a tag italic (<i>texto<i>), pois ela também não tem semântica.
